# قرآن أوفلاين (الرسم الإملائي) — حزمة جاهزة

هذه الحزمة تعطيك:
- اختيار السورة بالاسم (مع كتابة أول حرف/حرفين للتحديد تلقائيًا).
- اختيار الآية (وبالكتابة رقمياً).
- بحث بالنص (اكتب جزءاً من الآية → يحدد لك السورة والآية).
- يبني رابط Quran7m بصيغة: `http://www.quran7m.com/searchResults/SSSAAA.html`

> **مهم:** البحث بالنص يحتاج ملف `quran.json` (نص القرآن) موجود محلياً بجانب `index.html`.

---

## 1) تشغيل الموقع محليًا (بدون بايثون)

Windows يمنع أحيانًا قراءة JSON إذا فتحت الصفحة مباشرة `file:///...`.
الحل: شغّل سيرفر محلي بسيط عبر PowerShell (موجود في ويندوز):

1) افتح PowerShell داخل نفس المجلد.
2) نفّذ:

```powershell
powershell -ExecutionPolicy Bypass -File .\start_server.ps1
```

ثم افتح المتصفح على:
- http://localhost:8000

---

## 2) جلب ملف quran.json مرة واحدة (ثم يصبح كل شيء أوفلاين)

نفّذ:

```powershell
powershell -ExecutionPolicy Bypass -File .\download_quran.ps1
```

هذا السكربت يحاول تنزيل نص القرآن من أكثر من مصدر (ويطبع لك أي رابط نجح).
بعدها:
- ستجد `quran.json` داخل نفس المجلد
- افتح الموقع وسيعمل البحث أوفلاين ✅

> لو فشل مصدر، السكربت ينتقل تلقائيًا للمصدر التالي.

---

## 3) ملاحظات

- إذا أردت تغيير موقع الهدف (بدل Quran7m)، عدّل المتغير `Q7M_BASE` في `app.js`.
- صيغة الأرقام:
  - السورة 3 أرقام (054)
  - الآية 3 أرقام (011)
  - مع `.html` في النهاية

مثال: سورة القمر آية 11 ⇒ `054011.html`
